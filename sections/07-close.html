07 · Open items after source review

Questions to answer once—not repeatedly

CAD Guardian will first inspect the files and answer what can be resolved from evidence. Only the remaining decisions should return to Brian.

Project identification

Confirm the exact project record

  • Exact street address and project name.
  • Exact three sheet numbers, titles, and required content.
  • Which sketch/orientation is controlling where the iPad markup was flipped.
  • Which dimensions define the addition, deck, and any offset or jog conditions.
CAD environment

Confirm the production standard

  • AutoCAD version and required save-back version.
  • Controlling legacy DWG for preferred blocks and dimension styles.
  • Controlling current set for title block, paper space, fonts, CTB/STB, xrefs, and plotting.
  • Approved conventions for existing-to-remain, demolition, and new construction.
LiDAR and scale

Confirm the measurement basis

  • Scan file formats and whether the scans are already registered.
  • Expected file count and whether every scan has completed upload.
  • Which known dimensions or City floor-plate references should control scaling.
  • Whether any inaccessible or concealed areas require field verification or remain explicitly unresolved.
Commercial authorization

Confirm the production agreement

  • Fee or hourly structure after the estimate.
  • Desired first-review date and final delivery target.
  • Named BDPC reviewer and number of included consolidated review cycles.
  • Required final outputs: DWG, PDFs, xrefs, plot assets, and issue/source log.
Technology posture: use cutting-edge methods where they reduce manual effort and improve checking, but avoid “bleeding-edge” shortcuts that compromise dimensional integrity, file continuity, or BDPC control.
