04 · Floor-plan readiness

What makes floor-plan work responsible

I can produce floor plans. I should not pretend every source condition is equivalent. Schedule and confidence depend primarily on the quality of the source, required accuracy, and the amount of architectural interpretation expected.

Starting conditionTypical useEffort / riskResponsible position
Clean DWG/RVT + redlinesRevisions, permit comments, sheet updates, plan cleanupLowestBest immediate paid test. Scope one sheet or one bounded revision package.

Photos only / unknown scaleConcept referenceNot reliable for verified plansRequest dimensions, survey, scan, or field verification. Do not invent geometry.
Full permit set from incomplete inputsArchitectural production plus unresolved design decisionsNot responsibly estimableBreak into a diagnostic or sample sheet before committing to the full package.

Minimum source package I should request

  • Native files or the best available controlled background.
  • Survey, registered scan, or dimensioned markup establishing scale.
  • BDPC title block, template, layers/worksets, plot style, fonts, and example set.
  • Required plan content: rooms, doors, windows, dimensions, walls, fixtures, notes, schedules.
  • Accuracy tolerance and convention: face of finish, face of stud, centerline, or another standard.
  • Explicit exclusions, reviewer, review cycle, and issue format.

Output I can return for a bounded test

  • Native DWG or other agreed authoring file.
  • Review PDF using BDPC sheet and plotting standards.
  • Assumption, missing-information, and coordination issue log.
  • Tracked redline incorporation or revision summary.
  • One defined review cycle and corrected issue package.
  • Source/dependency inventory so the file can be opened and continued by BDPC.
Honest positioning: I am experienced in CAD production, manufacturing drawings, Autodesk automation, file governance, and technical delivery. I will not present myself as a licensed architect or as an architectural designer who should make independent project decisions. A bounded sample lets BDPC judge the actual production fit without either side overcommitting.
